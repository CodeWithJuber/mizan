# MIZAN Changelog

All notable changes to this project will be documented in this file.

## [v3.0.0] — 2025-01-01

### Added
- Agentic loop with ReAct pattern (up to 15 iterations per task)
- Multi-agent orchestration with Shura consensus protocol
- 7-layer Quranic Cognitive Architecture (QCA) pipeline
- Three-tier persistent memory system (episodic, semantic, procedural)
- Self-improvement through Hikmah learning and Nafs evolution
- Enterprise-grade security: JWT auth, rate limiting, command sandboxing
- Multi-channel gateway: Telegram, Discord, Slack, WhatsApp, Email
- Qadr automation engine with cron scheduling and webhook triggers
- Extensible skill registry with built-in skills
- React + Vite + Tailwind frontend with real-time WebSocket
- Docker Compose for dev and production with Nginx, SSL
- CLI interface: `mizan chat`, `mizan serve`, `mizan setup`
- One-line installer for macOS, Linux, Windows
- Production deployment script with Let's Encrypt SSL
